# المحاسب المحترف - Accounting Pro

## نظرة عامة
- **الاسم:** المحاسب المحترف (Accounting Pro)
- **الهدف:** منصة تعليمية تفاعلية باللغة العربية لتعلم إعداد القوائم المالية والتحليل المالي بالتطبيق العملي.
- **المميزات الرئيسية:**
  - مكتبة دروس تعليمية مفصّلة للمفاهيم المحاسبية (المعادلة المحاسبية، القيد المزدوج، القوائم المالية الأربع، النسب المالية).
  - مختبر عملي (Lab) يُدخل فيه المستخدم قيود اليومية، ويولّد آلياً: ميزان المراجعة، قائمة الدخل، قائمة المركز المالي، قائمة التغيرات في حقوق الملكية، قائمة التدفقات النقدية، والنسب المالية.
  - ثلاثة سيناريوهات جاهزة (شركة خدمات، شركة ناشئة، شركة تجارية ناضجة).
  - حاسبة نسب مالية سريعة مستقلة.
  - اختبار ذاتي تفاعلي مكوّن من 10 أسئلة.
  - محرك محاسبي **مُختبر بدقة (25/25 اختبار وحدوي)** يضمن صحة الحسابات.

## المزايا الفنية (Data & Architecture)
- **هياكل البيانات (Client-side فقط):**
  - `CHART_OF_ACCOUNTS`: دليل حسابات قياسي يحوي 55+ حساباً مصنفة بالكود والطبيعة (مدين/دائن) والقائمة المنتمية إليها (BS_A / BS_L / BS_E / IS_R / IS_X) والتصنيف الفرعي (متداول/غير متداول/بيعي/إداري...).
  - `JournalEntry`: قيد اليومية = (id, date, description, lines[]). كل سطر يحوي (accountCode, debit, credit).
  - `TrialBalance, IncomeStatement, BalanceSheet, EquityChanges, CashFlowStatement, Ratios`: هياكل مخرجة حسابياً من القيود.
- **التخزين:** لا يُستخدم تخزين دائم (التطبيق يعمل كـ SPA بالكامل في المتصفح). البيانات في حالة جلسة المستخدم (STATE object).
- **تدفق البيانات:**
  ```
  JournalEntries → TrialBalance → IncomeStatement → BalanceSheet → Ratios
                                                 ↘ EquityChanges
                                                 ↘ CashFlowStatement
  ```
- **محرك المحاسبة:** ينفذ مبدأ القيد المزدوج (Debit = Credit)، ويطبق معايير IAS 1 لهيكل قائمة الدخل والمركز المالي، ويحسب النسب الأكاديمية المعتمدة.

## الدقة المحاسبية
تم التحقق من دقة المحرك بـ **25 اختبار وحدوي** تغطي:
- ✅ التحقق من توازن القيد (Debit = Credit)
- ✅ رفض القيود غير المتوازنة
- ✅ دقة ميزان المراجعة
- ✅ صحة قائمة الدخل (صافي المبيعات، COGS، مجمل الربح، الربح التشغيلي، صافي الربح)
- ✅ تطبيق المعادلة المحاسبية (Assets = Liabilities + Equity)
- ✅ حساب النسب المالية الصحيح (التداول، السريعة، هوامش الربح)
- ✅ دقة عمليات الكسور (Float precision)
- ✅ معالجة الخصومات والمردودات
- ✅ تصنيف التدفقات النقدية (تشغيلي/استثماري/تمويلي)

جميع الاختبارات تُشغَّل تلقائياً في كل تحميل للصفحة، وتظهر النتائج في Console المتصفح (F12).

## دليل المستخدم

### 1. لتعلم الأساسيات
انتقل إلى قسم **"الدروس"** لقراءة شروحات عن:
- المعادلة المحاسبية الأساسية
- القيد المزدوج والرصيد الطبيعي
- القوائم المالية الأربع
- النسب المالية

### 2. للتطبيق العملي
انتقل إلى **"المختبر العملي"**:
1. اختر سيناريو جاهز (شركة خدمات / شركة ناشئة / شركة تجارية) من القائمة وحمّله، **أو** أدخل قيودك بنفسك.
2. انتقل بين التبويبات لرؤية التحولات التلقائية: اليومية → ميزان المراجعة → قائمة الدخل → المركز المالي → حقوق الملكية → التدفقات → التحليل المالي.
3. لاحظ علامات التحقق (✓ متوازن / ✗ غير متوازن) في كل مستوى.

### 3. لحساب نسب سريعة
استخدم **"حاسبة النسب"** لإدخال الأرقام الإجمالية مباشرة (بدون قيود) ورؤية جميع النسب مع التفسير.

### 4. لاختبار فهمك
خذ **"الاختبار الذاتي"** (10 أسئلة مع شرح تفصيلي لكل إجابة).

- **الحالة:** ✅ نشط
- **التقنيات:**
  - **Backend / SSR:** Hono Framework + TypeScript
  - **Build:** Vite + @hono/vite-build/cloudflare-pages

- حفظ القيود في Cloudflare KV/D1 مرتبطة بحساب المستخدم.
