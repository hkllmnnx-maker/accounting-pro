## نظرة عامة

**Accounting Pro** هو نظام محاسبي متكامل مصمم للشركات الصغيرة والمتوسطة، يوفر جميع الأدوات اللازمة لإدارة الحسابات والمبيعات والمشتريات والمخزون والتقارير المالية.

## الميزات الرئيسية

### إدارة العملاء والموردين
- إضافة وتعديل وحذف العملاء والموردين
- متابعة الأرصدة (مدين/دائن) لكل عميل ومورد
- عرض كشف حساب تفصيلي لكل جهة
- البحث السريع بالاسم

### إدارة المخزون والمنتجات
- قاعدة بيانات كاملة للمنتجات مع الباركود
- تتبع الكميات وأسعار الشراء والبيع
- **تنبيهات نفاد المخزون والمخزون المنخفض**
- فلترة المنتجات حسب حالة المخزون
- حساب قيمة المخزون الإجمالية
- تصنيف المنتجات حسب الفئات

### فواتير البيع والشراء
- إنشاء فواتير بيع وشراء احترافية
- إضافة عدة أصناف بكميات وأسعار مختلفة
- حساب تلقائي للخصم والضريبة
- تتبع المدفوع والمتبقي
- **البحث والفلترة حسب حالة الدفع**
- **طباعة ومشاركة الفواتير بصيغة PDF**

### السندات والمدفوعات
- سندات قبض وصرف
- ربط السندات بالعملاء والموردين
- طرق دفع متعددة (نقدي، تحويل، شيك)
- البحث في السندات

### المصاريف
- تسجيل وإدارة جميع مصاريف الشركة
- تصنيف المصاريف (إيجار، مرافق، صيانة، نقل، رواتب، أخرى)
- البحث والفلترة حسب التصنيف

### الموظفين والرواتب
- إدارة بيانات الموظفين
- تسجيل الرواتب والوظائف

### العروض والطلبات
- إنشاء عروض أسعار للعملاء
- تحويل العرض إلى فاتورة بنقرة واحدة
- تتبع حالة العروض

### القيود المحاسبية
- إدخال قيود يومية يدوية
- التأكد من توازن القيد (مدين = دائن)
- سجل كامل للقيود

### تحويل العملات
- تسجيل عمليات صرف العملات
- دعم عملات متعددة (USD, SAR, EUR, AED, EGP, وغيرها)

### التقارير المالية
- **لوحة تحكم تفاعلية** مع إحصائيات يومية وشهرية
- **تقارير المبيعات والمشتريات** برسوم بيانية
- **تقرير المصاريف** مع رسم دائري للتصنيفات
- **تقرير الأرباح والخسائر** مع رسم خطي للأرباح الشهرية
- **فلترة حسب الفترة الزمنية** (اليوم، الأسبوع، الشهر، السنة، مخصص)

### الإعدادات
- **الوضع الليلي والنهاري**
- بيانات الشركة (الاسم، العنوان، الهاتف، الرقم الضريبي)
- نسخ احتياطي واستعادة البيانات
- إعدادات اللغة والعملة

## التقنيات المستخدمة

- **Flutter 3.35.4** - إطار العمل
- **Dart 3.9.2** - لغة البرمجة
- **Hive** - قاعدة بيانات محلية سريعة
- **Provider** - إدارة الحالة
- **fl_chart** - الرسوم البيانية
- **pdf & printing** - توليد وطباعة ملفات PDF
- **share_plus** - مشاركة الملفات
- **intl** - التنسيق والترجمة

## البدء

- Flutter SDK 3.35.4+

```bash
# تنصيب الحزم
flutter pub get

# التشغيل على الويب
flutter run -d chrome

# بناء نسخة الإنتاج للويب
flutter build web --release

# بناء APK للأندرويد
flutter build apk --release
```
